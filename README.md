# MCR Bot

A powerful and cool macro bot for Geometry Dash 2.208, built with Geode SDK.

## Features
- **Record Bot**: Record your gameplay with precision.
- **Play Bot**: Replay your recorded macros.
- **Load/Save File**: Save your macros to `.mcr` files and share them.
- **Cross-Platform**: Supports Android, iOS, and Windows (via Geode).

## Credits
- Developed by **l4shdev**
- Built using [Geode SDK](https://geode-sdk.org/)

## Installation
1. Install [Geode](https://geode-sdk.org/install).
2. Download the `.geode` file from the releases page (coming soon).
3. Drag and drop the file into your Geometry Dash game or use the Geode mod menu to import.

## How to Use
1. Open any level.
2. Pause the game.
3. Click the MCR button (Play icon in a circle).
4. Use the menu to Record, Play, Save, or Load your macros.

## Development
To build this mod yourself, you need the Geode SDK installed.
```bash
geode build
```
